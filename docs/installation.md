# Installation

## Stable release

To install Homogenization for climate, run this command in your terminal:

```sh
uv add homogenization
```

Or if you prefer to use `pip`:

```sh
pip install homogenization
```

## From source

The source files for Homogenization for climate can be downloaded from the [Github repo](https://github.com/karenamckinnon/homogenization).

You can either clone the public repository:

```sh
git clone git://github.com/karenamckinnon/homogenization
```

Or download the [tarball](https://github.com/karenamckinnon/homogenization/tarball/master):

```sh
curl -OJL https://github.com/karenamckinnon/homogenization/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd homogenization
uv pip install .
```
